# SMARTSTOCK
SmartStock System adalah aplikasi manajemen stok barang yang dirancang khusus untuk membantu bisnis kecil dan UMKM dalam mengatur persediaan secara lebih mudah, cepat, dan efisien. Sistem ini dibangun menggunakan konsep Object-Oriented Programming (OOP) sehingga struktur program menjadi lebih rapi, modular, dan mudah dikembangkan.
